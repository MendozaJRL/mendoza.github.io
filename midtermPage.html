Midterm Period

Having multiple virtual machines, we have faced difficulties concerned with the redundant and inefficient deployment of installing applications and services that are required on each remote nodes. To solve these, the introduction with ansible playbooks were necessary in order to further simplify tasks.

With ansible playbook, deployment of tasks were much easier as it only requires you to create a YAML file which contains the tasks and the target nodes. In addition, ansible roles were also useful as it further organizes the infrastructure in which, the specific tasks are imported from the ansible role only when needed without changing the entirety of the main playbook file.

Furthermore, services such as log monitoring, performance monitoring, and availability monitoring tools; are very useful in managing remote servers. Since we are working in a DevOps environment, we are required to serve the damands of the consumers, as such 100% uptime and fast debugging should be done. With this, the monitoring tools will help the system administrator to determine where the problem is and address it ahead of time.
