Brief Introduction

Managing Enterprise Servers

DevOps

Integration of Development and Operations such that the program updates, bug fixes, deployment, managing, and operation on the servers are working simultaneously as one. DevOps helps on having automated and efficient workflows, better infrastructure, continuous management, and allowing one hundred percent uptime of the application.

Ansible

Ansible is a project sponsored by Red Hat. It lets the system administrator to deploy a multitude of tasks and services efficiently. Instead of manual deployment, Ansible automates the configurations and orchestration by simply writing tasks on a playbook.

GitHub

It is a platform that allows users to collaborate on working with projects across different remote areas. It also allows the user to safely modify the code and recover them. It uses repositories to host collaborative workspace which allows the user to import their code (push), export (pull), and observe through the use of branches (version control).

Significant Learnings and Reflection

Conclusion

Managing multiple hosts is very crucial to do without the help of automation. As the number of hosts increases, the more work is needed to be done by the system administrator. With the help of various infrastructures, platforms, and services; system administrators are able to deploy commands and tasks at one execution.

Since we are now in a technological world, meeting the consumer's demands is necessary. Moreover, applications and services should be developed and operated continuously. In addition, a collaborative workspace must also be observed in order to meet these expectations. By understanding how GitHub, Ansible, DevOps, Virtualization, and Containerization works; we as students would have a brief idea on system administrator's role on ensuring that every demand is met and is delivered successfully.
